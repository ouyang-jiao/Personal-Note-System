# 个人笔记与知识库系统

基于 Vue 3 构建的现代化个人笔记管理系统，集成多种前端技术栈，提供完整的笔记管理、分类标签、数据统计等功能。



**在线预览**: [https://magenta-bombolone-6b1b75.netlify.app](https://magenta-bombolone-6b1b75.netlify.app)

## ✨ 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue | 3.4+ | 渐进式 JavaScript 框架 |
| Vue Router | 4.3+ | 路由管理 |
| Pinia | 2.1+ | 状态管理 |
| Element Plus | 2.6+ | UI 组件库 |
| ECharts | 6.1+ | 数据可视化 |
| Vite | 5.2+ | 构建工具 |

## 🚀 功能特性

### 核心功能

- **Vue 双向绑定** - 使用 v-model 实现表单数据与视图的实时同步
- **状态持久化** - 通过 localStorage 保存所有数据，刷新不丢失
- **路由管理** - 多页面路由配置，支持路由守卫和权限控制
- **组件化开发** - 模块化组件设计，提高代码复用性

### 笔记管理

- ✅ 笔记新增/编辑（标题、内容、分类、标签）
- ✅ 实时表单校验（Element Plus 表单校验组件）
- ✅ 笔记搜索（关键词模糊搜索、按分类/标签筛选）
- ✅ 笔记列表卡片渲染（v-for 循环展示）
- ✅ 分页组件（Element Plus 分页）
- ✅ 排序功能（按创建时间、修改时间、置顶优先）
- ✅ 笔记置顶、删除、移入回收站

### 分类与标签

- ✅ 分类管理（新增、编辑、删除）
- ✅ 标签管理（批量管理、新增/删除）
- ✅ 笔记多标签绑定
- ✅ 分类侧边栏导航

### 用户认证

- ✅ 用户注册
- ✅ 用户登录（支持记住密码）
- ✅ 修改密码
- ✅ 退出登录

### 数据看板

- ✅ 笔记总数统计
- ✅ 各分类笔记占比（饼图）
- ✅ 月度新增笔记曲线（折线图）
- ✅ 标签使用频次（柱状图）

### 系统设置

- ✅ 主题切换（浅色/深色模式）
- ✅ 修改密码
- ✅ 数据统计展示

## 📁 项目结构

```
src/
├── components/          # 公共组件
│   ├── Header.vue       # 顶部导航栏
│   └── Sidebar.vue      # 侧边栏导航
├── router/
│   └── index.js         # 路由配置
├── stores/
│   ├── auth.js          # 用户认证状态
│   └── note.js          # 笔记数据状态
├── styles/
│   └── global.css       # 全局样式
├── views/               # 页面组件
│   ├── Login.vue        # 登录页
│   ├── Register.vue     # 注册页
│   ├── Home.vue         # 首页（知识库总览）
│   ├── AddNote.vue      # 新建笔记
│   ├── EditNote.vue     # 编辑笔记
│   ├── NoteDetail.vue   # 笔记详情
│   ├── Category.vue     # 分类管理
│   ├── Tag.vue          # 标签管理
│   ├── Recycle.vue      # 回收站
│   ├── Dashboard.vue    # 数据看板
│   └── Settings.vue     # 系统设置
├── App.vue              # 根组件
└── main.js              # 入口文件
```

## 📝 使用说明

1. **登录系统**: 使用默认账户或注册新账户登录
2. **创建笔记**: 点击"新建笔记"，填写标题、内容，选择分类和标签
3. **管理分类**: 在"分类管理"中添加或编辑分类
4. **管理标签**: 在"标签管理"中添加或删除标签
5. **查看统计**: 进入"数据看板"查看笔记统计信息
6. **设置主题**: 在"系统设置"中切换浅色/深色模式
