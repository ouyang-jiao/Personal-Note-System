Vue 双向绑定

核心原理
Vue 3 使用 v-model 指令实现数据与视图的双向绑定，基于 Proxy 代理对象，当数据变化时自动更新视图，视图变化时自动更新数据。
<template>
<el-form :model="form">
<el-input v-model="form.title" />
<el-textarea v-model="form.content" />
</el-form>
</template>
用户输入
↓
v-model 捕获事件
↓
更新响应式数据
↓
视图自动更新

状态保存（localStorage）

实现方式
使用 localStorage 实现数据持久化，将笔记、分类、标签、用户等数据存储在浏览器本地，刷新页面后数据不丢失。
function saveNote(note) {
const notes = JSON.parse(localStorage.getItem('notes')) || []
notes.push(note)
localStorage.setItem('notes', JSON.stringify(notes))
}
💾
数据持久化
所有数据自动保存到 localStorage，刷新页面不丢失
🔄
自动同步
数据变化时自动同步到本地存储，无需手动保存

路由管理（Vue Router）

路由配置
使用 Vue Router 4 配置多页面路由，支持路由守卫控制访问权限，实现用户认证后的页面访问控制。
const routes = [
{ path: '/login', meta: { requiresAuth: false } },
{ path: '/', meta: { requiresAuth: true } },
{ path: '/add', meta: { requiresAuth: true } },
]

router.beforeEach(to => {
if (to.meta.requiresAuth && !isLoggedIn) {
return '/login'
}
})
  • 路由守卫：控制未登录用户访问受限页面
  • 动态路由：根据登录状态显示不同导航
  • 路由元信息：标记页面是否需要认证

项目亮点与技术难点

🌟
技术亮点
完整的前后端一体化解决方案，使用 Vue 3 最新技术栈，支持深色模式切换，提供丰富的动画效果和交互体验。
⚡
技术难点
路由守卫的权限控制逻辑、Pinia 状态与 localStorage 的同步、Element Plus 组件在深色模式下的样式覆盖。
🚀
解决方案
通过路由元信息实现权限控制，在 Pinia 的 actions 中统一处理 localStorage 同步，使用 CSS 变量和深度选择器覆盖组件样式。
